- Designed and built S_Entry_Exit_Indicator.ts from scratch as a clean replacement for daily timeframe individual stock trading
- File committed and ready to paste into thinkorswim Script Editor

**Key design decisions:**
- **No EMA ribbon** — ZigZag-only approach; EMAs are trend-following and add lag for reversal detection
- **Symmetric pivot detection:** `high[pivotBars] == Highest(high, 2*pivotBars+1) AND high[pivotBars] > Highest(high, pivotBars)` — the second condition prevents tie-matching the current bar as a false confirmation
- **Signal fires pivotBars bars after the actual swing** (no future-painting). Entry = close of the confirmation bar
- **Stop** = swing bar price ± ATR(14) × 0.25 buffer (anchored to the actual swing, not entry)
- **Target** = entry close ± (prior opposite-pivot distance × 1.618) — anchored at entry for cleaner R:R visibility
- **`rec` variables** (`lastSignal`, `entryLevel`, `stopLevel`, `targetLevel`) latch the most recent signal's levels and persist until the next signal fires
- **`targetLevel > 0` guard** prevents a zero-line before a valid target exists (before first opposite pivot is seen)
- **`active = lastSignal != 0`** guards all plots so nothing draws before the first signal

**Visual output:** green/red arrows at confirmation bar; white/red/green dashed horizontal lines; right-edge price bubbles; LONG SIGNAL/SHORT SIGNAL/WAITING status label; green/red/gray bar coloring

**Inputs:** pivotBars=5, atrLength=14, atrBuffer=0.25, fibTarget=1.618, showLabels=yes, showStop=yes, showTarget=yes
